¿Como Jugar?

Usa las flechas para mover las celdas.

Objetivo:
Cuando el valor de dos celdas coincidan, se fusionarán sumando ambos valores.
El objetivo es llegar a 2048 (o más... 😜)

Reglas:
En cada movimiento, se moveran y fusionaran todas las celdas en la direccion indicada
Si no quedan movimientos posibles GAME OVER!
